 ☕ Cafe Menu Project

This is a simple and attractive Cafe Menu web application developed using HTML, CSS, and PHP. The project is designed to display menu items and calculate the bill dynamically based on user selection.

## 🔥 Features
- 📋 Attractive cafe menu design
- 🧾 Dynamic bill generation
- 🍔 Multiple food items (coffee, burger, sandwich, ice cream, etc.)
- 🎨 Styled using CSS for better UI
- 🔗 Navigation between pages (Menu & Bill)

## 🛠️ Technologies Used
- HTML
- CSS
- PHP

## 📂 Project Structure
- `index.php` → Home page
- `menu.php` → Menu display
- `bill.php` → Bill calculation
- `style.css` → Design styling
- `images/` → Image assets
